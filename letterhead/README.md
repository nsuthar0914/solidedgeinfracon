# SolidEdge Letterhead

Files in this folder:

- `solidedge-letterhead-a4.png`: full-page A4 letterhead preview at 300 DPI
- `solidedge-letterhead-header.png`: header artwork for Google Docs
- `solidedge-letterhead-footer.png`: footer artwork for Google Docs

Recommended Google Docs setup:

1. Open your Google Doc.
2. Go to `Insert` -> `Headers & footers` -> `Header`.
3. Insert `solidedge-letterhead-header.png` into the header and resize it to page width.
4. Go to `Insert` -> `Headers & footers` -> `Footer`.
5. Insert `solidedge-letterhead-footer.png` into the footer and resize it to page width.
6. Set document margins so the body text sits comfortably between the header and footer.

Suggested page settings:

- Paper size: `A4`
- Top margin: `1.2 in`
- Bottom margin: `1.0 in`
- Left margin: `1.0 in`
- Right margin: `1.0 in`
